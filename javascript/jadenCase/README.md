JadenCase 문자열 만들기
=============================

#1. 이해
-----------------------------
문자열 s가 주어졌을 때, 모든 단어의 첫 문자가 대문자이고 그 외의 알파벳은 소문자가 되도록 변경하여 리턴하라.

#2. 계획
-----------------------------
공백을 기준으로 현재의 문자가 단어의 첫 문자인지 판별하여 대문자로 바꾸거나 소문자로 바꾼다.

-----------------------------

#4. 회고
-----------------------------
공백이 여러 개 연속해서 주어지는 경우의 수도 생각하여야 했다.
