정수 제곱근 판별
====================

#1. 이해
------------------
자연수 n이 주어질 때, n의 제곱근 m이 존재한다면 m+1의 제곱을 리턴하고, 존재하지 않으면 -1을 리턴하라.

#2. 계획
---------------
sqrt 함수를 통해 n의 제곱근 m을 구하고, m이 정수가 아닌 경우 -1를 리턴하고 정수인 경우 m+1의 제곱을 리턴한다.
