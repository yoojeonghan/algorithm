제일 작은 수 제거하기
=============================

#1. 이해
-----------------------------
배열에서 가장 작은 수를 제거하고 리턴하라. 만약 리턴하려는 배열이 1인 경우에는 -1을 넣어 리턴하라.

#2. 계획
-----------------------------
배열을 순회하며 가장 작은 값을 찾은 뒤, filter함수를 통하여 해당 요소를 제거한 배열을 리턴한다.
