피보나치 수
====================

#1. 이해
------------------
피보나치 수는 F(0) = 0, F(1) = 1일 때, 1 이상의 n에 대하여 F(n) = F(n-1) + F(n-2) 가 적용되는 수이다. 2이상의 n이 입력되었을 때, n번째 피보나치 수를 1234567으로 나눈 후 나머지 값을 리턴하라.

#2. 계획
---------------
반복문으로 n까지 순회하여, 피보나치 수를 구한 값을 배열에 넣은 후 n번째 값을 리턴한다.
