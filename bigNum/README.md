큰 수 만들기
====================

#1. 이해
------------------
숫자로 이루어진 문자열 number와 정수 k가 주어질 때, 두 개의 숫자를 제거하면 만들 수 있는 가장 큰 숫자를 구하여라.

#2. 계획
---------------
number 배열을 순회하며 현재 인덱스가 다음 인덱스보다 작다면 문자열에서 제거하고 k를 1 감소시킨다. k가 0이 되면 루프를 종료한다. 루프 종료 후 k가 0보다 크다면 그 만큼 문자열에서 제거한다.
