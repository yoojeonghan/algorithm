H-Index
====================

#1. 이해
------------------
논문 인용 횟수 h가 적힌 배열 n개 중, h 이상의 수가 h개 이상이고, 나머지가 h개 이하일 때 h의 최댓값을 구하여라.

#2. 계획
---------------
Map에 인용 횟수와 논문의 수를 쌓은 뒤, 논문의 수 순으로 내림차순 정렬한다. 인용 횟수와 논문의 수가 같은 경우 answer에 넣고, answer가 빈 값인 경우 가장 논문의 수가 많은 논문의 인용 횟수를 answer에 넣는다.
